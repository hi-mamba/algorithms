# 把二叉树打印成多行

## 题目描述

从上到下按层打印二叉树，同一层结点从左至右输出。每一层输出一行。

例如:
```
给定二叉树: [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7
返回其层次遍历结果：

[
  [3],
  [9,20],
  [15,7]
]
```
 
## 思路

按`层次`输出二叉树

访问根节点，并将根节点入队。

当队列不空的时候，重复以下操作。
```
弹出一个元素。作为当前的根节点。
如果根节点有左孩子，访问左孩子，并将左孩子入队。
如果根节点有右孩子，访问右孩子，并将右孩子入队。
```
